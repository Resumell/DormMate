# 3D外壳设计说明

## 当前状态

外壳设计尚未完成，本目录预留用于存放外壳设计文件。

## 设计要点

1. **材料**：建议 PLA 或 ABS 3D打印
2. **尺寸**：适配 ESP32 开发板 + 传感器模块 + OLED 屏幕
3. **开孔**：
   - OLED 屏幕窗口
   - DHT11 温湿度传感器通风口
   - 光线传感器透光孔
   - PIR 红外传感器检测窗口
   - 语音模块出声孔
   - 电源 / USB 接口开口
4. **散热**：预留散热格栅
5. **壁挂**：支持墙面安装或桌面放置
